# Todo → Google Calendar 自動スケジューラー

Google Sheetsに書いたTodoを、毎朝4:30に自動でGoogleカレンダーの空き枠に登録するスクリプトです。

## 仕組み

```
毎朝4:30（GitHub Actions）
    ↓
Google Sheetsからタスク一覧を読み込む
    ↓
Google Calendarの今日の空き時間を取得
    ↓
優先順位ロジックで割り当て
  └─ 午前 → 思考系タスク（重要度高）を優先
  └─ 午後 → 単純作業タスクを割り当て
    ↓
所要時間 × 1.3（バッファ30%増し）でブロック
    ↓
絵文字つきタイトルでGoogleカレンダーに登録
```

## Google Sheetsの列構成

| 列 | 内容 | 例 |
|---|---|---|
| A | タスク名 | プレゼン資料の最終確認 |
| B | 重要度 | 高 / 中 / 低 |
| C | 緊急度 | 高 / 中 / 低 |
| D | タスク種別 | 思考系 / 単純作業 |
| E | 所要時間（分） | 60 |
| F | 期日 | 2026/03/15 |
| G | ステータス | 未着手 / 進行中 / 完了 |
| H | メモ | （任意） |

## 絵文字ルール（自動付与）

| 条件 | 絵文字 |
|---|---|
| 重要度：高 × 緊急度：高 | 🔴 |
| 重要度：高 × 緊急度：中以下 | 🌟 |
| 重要度：中 × 緊急度：高 | 🟡 |
| 重要度：低 | 🔵 |
| タスク種別：思考系 | 🧠（上記に追加） |
| タスク種別：単純作業 | ✅（上記に追加） |

例: `🔴🧠 プレゼン資料の最終確認`

## スケジューリングルール

- **バッファ**: 所要時間 × 1.3 でカレンダーブロック（例: 60分タスク → 78分確保）
- **午前（9:00〜12:00）**: 思考系タスクを重要度順に割り当て
- **午後（13:00〜18:00）**: 単純作業タスクを期日順に割り当て
- **空き枠が足りない場合**: 翌日以降に繰り越し（スキップしない）
- **既に登録済みのタスクは重複登録しない**

## セットアップ手順

詳細は `SETUP.md` を参照してください。

1. Google Cloud Console でプロジェクト作成・API有効化
2. `credentials.json` を取得
3. Google Sheetsのシート名・IDを設定
4. GitHub Secretsに認証情報を登録
5. GitHub Actionsが毎朝4:30（JST）に自動実行
